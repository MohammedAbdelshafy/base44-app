# revenue-review

**Goal:** Ensure we only ship value that makes money.
**Action:**
1. Read the artifacts generated by the Evidence Collector.
2. Audit the data against the mission's KPI (e.g., BAGA's "Send me another batch" criteria).
3. Remove duplicates, verify contacts, and clean the final deliverable.
4. Output the final, polished CSV/Report to `MBM/Clients/`.
**Rule:** If the data will not confidently generate revenue, reject it and send it back to the collector.
