# mission-planner

**Goal:** Transparent planning and deterministic routing.
**Action:**
1. Receive high-level objective (e.g. from a ChatGPT proxy).
2. Break down the objective into an atomic, deterministic checklist.
3. Hand off execution tasks to the Evidence Collector or Revenue Review skills.
**Rule:** No vague goals. Create a concrete plan artifact for every mission so the execution layer knows exactly what to do.
