# Style Guide — Don't Watch This

- **Voice:** suspense-heavy, dark, mysterious
- **Visual style:** dark visuals, high contrast, desaturated
- **Emoji policy:** minimal
- **Hook style:** suspense-heavy
- **Theme:** dark stories, chilling truths, mystery, suspense

Every clip, title, caption, and thumbnail must read as a Don't Watch This asset, not a generic MBM clip.
